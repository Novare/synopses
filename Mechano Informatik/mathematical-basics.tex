\subsection{Quaternions short}%
\label{mat:sub:quaternione_short}
\[ q = (a, b, c, d) = a + bi + cj + dk\]
\[ ijk = -1,\ ij = k,\ jk = i,\ ki = j\]
Einheitsquaternionen
\[ ||q|| = 1\]
Konjugierte
\[ q* = (a, -b, -c, -d)\]
Inverse
\[ q^{-1} = \frac{q^*}{||q||}\]
Rotationen immer nur mit Einheitsquaternionen!\\
Rotation:
\[ p' = qpq^*\]
